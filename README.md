# FinFlow

FinFlow is a simple finance management website made using HTML and CSS.

## Features

- Home Page
- Product Page
- About Page
- Contact Page
- Responsive layout
- Clean and simple design

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
FinFlow/
│── index.html
│── product.html
│── about.html
│── contact.html
│── style.css
│── images/
```

## How to Run

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Open `index.html` using Live Server or any web browser.
